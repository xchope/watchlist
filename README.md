# MyWatchlist — Milestone 1

Kurzbeschreibung (Thema):
>Eine einfache Watchlist für Filme/Serien. Nutzer:innen können Titel zur Liste hinzufügen, als „gesehen“ markieren und Bewertungen/Notizen speichern. Einstellungen/Listen sollen beim Verlassen der Seite erhalten bleiben.


Aufruf im Browser:  
`http://localhost:8080/api/watchlist`
